# Briseyda’s Personal Website

This project is my personal webpage created for Assignment 02. It demonstrates basic HTML structure, CSS styling, and the use of GitHub Pages for hosting a live website. The purpose of this site is to practice building and publishing a simple webpage while learning version control and project documentation.

## 🌐 Live Website
https://brie95.github.io

## 📁 Project Structure
- **index.html** — Main webpage content  
- **style.css** — Custom styling for layout, colors, and typography  
- **README.md** — Project overview and documentation  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- GitHub Pages  
- Git & GitHub for version control  

## 📘 What I Learned
- How to structure and style a basic webpage  
- How to create and manage a GitHub repository  
- How to publish a live website using GitHub Pages  
- How to write clear commit messages and maintain a version‑controlled workflow
