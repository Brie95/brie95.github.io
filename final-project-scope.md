# Project Scope Statement – Project 03 Final Project  
**Project Manager:** Brie Lopez  
**Date:** April 21, 2026  
**Project Title:** Dodo’s Coffee Bar  

---

## 1. Project Purpose
The purpose of this project is to design and develop a multi‑page website for **Dodo’s Coffee Bar**, a fictional neighborhood café known for its cozy atmosphere, handcrafted drinks, and community‑centered vibe. This project demonstrates my ability to apply PMUPM Chapter 3 principles by defining a clear scope, establishing boundaries, and planning a structured web project from concept to completion. The final website will serve as a portfolio‑ready artifact showcasing both technical and project‑management skills.

---

## 2. Project Description
This project involves creating a themed website representing Dodo’s Coffee Bar. The site will highlight the café’s brand identity, menu offerings, story, and customer experience. It will include multiple HTML pages, consistent CSS styling, images, and intuitive navigation.

The brand aesthetic will reflect a **warm, earthy palette** inspired by natural wood, espresso tones, and soft ambient lighting. Typography and layout choices will reinforce the café’s welcoming, community‑oriented personality. The website will be published using GitHub Pages and will reflect intentional design choices aligned with the fictional business’s identity.

---

## 3. Desired Results
The project will be considered successful when the following outcomes are achieved:

- A fully functional, multi‑page website representing Dodo’s Coffee Bar  
- A cohesive visual style that reflects the café’s warm, inviting brand  
- Clear, organized content that communicates the business’s offerings and identity  
- A functional navigation system connecting all pages  
- A live GitHub Pages deployment accessible via a public URL  
- Documentation demonstrating planning, decision‑making, and alignment with PMUPM principles  

---

## 4. Project Deliverables

### **Website Deliverables**
- `index.html` (homepage introducing Dodo’s Coffee Bar)  
- Additional pages such as:  
  - Menu (coffee, pastries, seasonal drinks)  
  - About Us (story, mission, values)  
  - Gallery (ambiance photos)  
  - Contact (hours, location, contact info)  
- `style.css` with consistent branding and layout  
- Images, icons, and assets supporting the café theme  
- A published GitHub Pages site  

### **Documentation Deliverables**
- This Project Scope Statement (`final-project-scope.md`)  
- Any additional planning notes required by the assignment  

---

## 5. Project Exclusions
To prevent scope creep and maintain focus, the following items are not included:

- Backend systems (ordering, reservations, inventory, login, etc.)  
- JavaScript‑based interactivity beyond assignment requirements  
- E‑commerce functionality  
- Advanced animations, frameworks, or libraries  
- Real‑world business operations or actual customer data  
- Any features not explicitly listed in the deliverables section  

---

## 6. Constraints
- **Time:** Must be completed by the Project 03 deadline  
- **Tools:** Limited to HTML, CSS, GitHub Pages, and course‑approved resources  
- **Quality:** Must meet rubric standards for design, structure, and functionality  
- **Resources:** All assets must be created or sourced by the student  
- **Brand Consistency:** Colors, typography, and layout must remain consistent across all pages  

---

## 7. Assumptions
- All images and branding elements will be fictional or royalty‑free  
- GitHub Pages will remain available for deployment  
- The project requirements will remain stable after this scope is approved  
- I will have consistent access to my development tools and internet connection  

---

## 8. Acceptance Criteria
The project will be accepted when:

- All required pages are complete, functional, and visually consistent  
- Navigation works across all pages  
- The website reflects the branding and personality of Dodo’s Coffee Bar  
- The site is published and accessible via GitHub Pages  
- All documentation is complete and submitted on time  
- The final product aligns with this scope statement and the Project 03 rubric
